# Early Developmemnt of Semantic News Site (NLP) with Async Discord
## Django, GraphQL, JWT-Auth, ApacheSpark-NLP, Celery, Redis Docker
Sources news artilces with async ETL pipeline. 
Provides an async discord to share news. 
In addition to boolean operators and keywords, users can search for articles semantically. 
Objective: Provide a flipboard/discord clone that incorporates NLP analysis and SVG recommender system.
